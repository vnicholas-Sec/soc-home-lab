# soc-home-lab
Building and documenting a personal SOC home lab to develop threat detection and incident response skills.
SOC Home Lab

## Project Overview

This repository documents the process of building a personal Security Operations Center (SOC) home lab from scratch.

The purpose of this project is to develop practical skills in:

- Threat detection
- Log analysis
- SIEM fundamentals
- Incident investigation
- Blue Team methodologies

This is an ongoing project and will be continuously updated as the lab environment evolves.

## Planned Lab Architecture

The lab will include:

- VirtualBox (Hypervisor)
- Kali Linux (Attacker Machine)
- Windows 10 (Target Machine)
- Sysmon (Enhanced logging)
- Splunk Free (SIEM platform)

Network setup:
- Internal virtual network for controlled attack simulation

---

## Learning Goals

- Understand how attacks generate logs
- Analyze Windows Event Logs
- Monitor authentication failures (Event ID 4625)
- Track process creation events (Sysmon Event ID 1)
- Map attack techniques to MITRE ATT&CK framework
- Build detection logic and alerting strategies
